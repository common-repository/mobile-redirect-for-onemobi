This plugin will let you redirect your website to your onemobi mobile website
 
== Description ==
 
This plugin will use the onemobi API to let you redirect your website to your onemobi mobile website.
 
== Installation ==
 
1. Upload the plugin to `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Input your API settings under Settings > onemobi Redirect
 
== Frequently Asked Questions ==
  
= What is onemobi? =
 
The onemobi mobile website builder lets you make a website that functions perfectly on any mobile device. It can work as a stand alone website or can piggyback onto your existing traditional website. A line of code is simply inserted on your traditional website and people accessing it via mobile are automatically redirected to the mobile-ready website.

= Where can I get my API Key =
 
Login to your onemobi control panel and then click Settings in the top bar. Look under Account Prefrences. Your API key will be in this format: XXXX-XXXX-XXXX-XXXX-XXXX-XXXX-XXXX-XXXX
